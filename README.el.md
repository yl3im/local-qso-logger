Ένας καταγραφέας QSO ραδιοερασιτεχνών που σέβεται την ιδιωτικότητα και εκτελείται εξ ολοκλήρου στον περιηγητή σας. Χωρίς λογαριασμό, χωρίς διακομιστή, χωρίς παρακολούθηση, χωρίς αναλυτικά — τα ημερολόγιά σας υπάρχουν μόνο στο `localStorage` του περιηγητή σας και δεν φεύγουν ποτέ από τη συσκευή σας.

Από τον [YL3IM](https://www.qrz.com/db/YL3IM). Ιστότοπος του έργου: [qso.ham.lv](https://qso.ham.lv).

## Δυνατότητες

- Πολλαπλά ημερολόγια, καθένα με τη δική του λίστα QSO.
- Ενέργειες ημερολογίου: δημιουργία, μετονομασία, διαγραφή, εισαγωγή από ADIF, εξαγωγή σε ADIF (`.adi`).
- Πεδία QSO: διακριτικό, ημερομηνία UTC, ώρα UTC, μπάντα, τρόπος, RST απεσταλμένο, RST ληφθέν.
- Επεξεργασία και διαγραφή οποιουδήποτε QSO (με επιβεβαίωση στη διαγραφή).
- Λογικές προεπιλογές: σημερινή ημερομηνία/ώρα UTC προσυμπληρωμένες, προεπιλογές RST βάσει τρόπου (59 για φωνητικούς τρόπους, 599 για CW/ψηφιακούς), επίμονη μπάντα και τρόπος μεταξύ συνεχόμενων QSO.
- Ζωντανός δείκτης διπλότυπου διακριτικού (ενημερωτικός — τα διπλότυπα επιτρέπονται).
- Στήλη σημαίας χώρας που εξάγεται από το πρόθεμα του διακριτικού (καλύπτει ≥99 % των συνηθισμένων προθεμάτων ραδιοερασιτεχνισμού, συμπεριλαμβανομένων φορητών διακριτικών όπως `9A/M0NCG`).
- Εμφάνιση ημερομηνίας με γνώση του locale στον πίνακα QSO· η αποθήκευση ISO και η έξοδος ADIF παραμένουν αμετάβλητες.
- Θέματα ημέρας/νύχτας (ημέρα προεπιλεγμένη· ο διακόπτης βρίσκεται στην κεφαλίδα).
- Φιλική προς κινητά responsive διάταξη με κουμπιά κατάλληλα για αφή.
- Λειτουργεί εντελώς εκτός σύνδεσης — καθόλου αιτήματα δικτύου σε καμία στιγμή.
- Εγκαταστάσιμο ως PWA (Προσθήκη στην οθόνη Αφετηρίας / Εγκατάσταση εφαρμογής) όταν φιλοξενείται μέσω HTTPS.
- Διεπαφή διαθέσιμη σε **28 γλώσσες** (αγγλικά συν 22 λατινικές, 5 κυριλλικές και ελληνικά)· επιλογέας με emoji σημαιών στην κεφαλίδα.

## Ξεκινώντας

Απλώς ανοίξτε το `index.html` σε έναν σύγχρονο περιηγητή. Χωρίς build, χωρίς εγκατάσταση, χωρίς διακομιστή.

Για να το φιλοξενήσετε, τοποθετήστε τα στατικά αρχεία (`index.html`, `style.css`, `app.js`, `favicon.svg`, `manifest.webmanifest`, `service-worker.js` και τον κατάλογο `i18n/` με τα 28 αρχεία μεταφράσεων) σε οποιονδήποτε στατικό φορέα φιλοξενίας (GitHub Pages, Netlify, δικός σας web διακομιστής). Θα λειτουργεί επίσης μέσω `file://` — η εγγραφή του service worker παρακάμπτεται αυτόματα για το πρωτόκολλο `file:`, οπότε το άνοιγμα του `index.html` απευθείας από το δίσκο λειτουργεί καθαρά.

Όταν παρέχεται μέσω HTTPS, η εφαρμογή γίνεται εγκαταστάσιμη ως PWA (μέσω του μενού *Εγκατάσταση εφαρμογής* / *Προσθήκη στην οθόνη Αφετηρίας* του περιηγητή) και λειτουργεί εκτός σύνδεσης μετά την πρώτη επίσκεψη χάρη σε έναν cache-first service worker που προ-φορτώνει κάθε στατικό αρχείο (UI + όλες τις μεταφράσεις).

Ένα προεπιλεγμένο ημερολόγιο δημιουργείται αυτόματα στην πρώτη επίσκεψη, ώστε να μπορείτε να ξεκινήσετε την καταγραφή αμέσως.

## Ημερολόγια

- **Δημιουργία:** πληκτρολογήστε ένα όνομα στο πεδίο *Όνομα ημερολογίου* και υποβάλετε. Αν αφήσετε το όνομα κενό, η προεπιλογή θα είναι `Log YYYY-MM-DD HH:MM UTC`.
- **Εναλλαγή:** κάντε κλικ σε οποιοδήποτε ημερολόγιο στην πλαϊνή γραμμή.
- **Μετονομασία:** κάντε κλικ στο *Μετονομασία* στην κεφαλίδα του ημερολογίου. Το Enter αποθηκεύει, το Escape ακυρώνει.
- **Διαγραφή:** κάντε κλικ στο *Διαγραφή ημερολογίου*. Θα σας ζητηθεί επιβεβαίωση. Αν διαγράψετε το τελευταίο ημερολόγιο, ένα νέο δημιουργείται αυτόματα.

## QSO

- Συμπληρώστε τη φόρμα και πατήστε **Καταγραφή QSO**.
- Το διακριτικό μετατρέπεται αυτόματα σε κεφαλαία καθώς πληκτρολογείτε.
- Η ημερομηνία και η ώρα προ-συμπληρώνονται σε *τώρα* σε UTC και επαναφέρονται μετά από κάθε καταγεγραμμένο QSO· μπορείτε ακόμα να εισαγάγετε οποιαδήποτε τιμή.
- Η μπάντα και ο τρόπος διατηρούνται μεταξύ QSO στην ίδια συνεδρία, ώστε να μη χρειάζεται να τα επιλέξετε ξανά για κάθε επαφή.
- RST απεσταλμένο / RST ληφθέν, αν αφεθούν κενά, επανέρχονται σε **59** για φωνητικούς τρόπους (SSB/FM/AM/DIGITALVOICE) και σε **599** για CW και ψηφιακούς τρόπους (CW/FT8/FT4/RTTY/PSK31/JT65).
- Ένα chip *Διπλότυπο σε αυτό το ημερολόγιο* εμφανίζεται κάτω από το πεδίο του διακριτικού αν το διακριτικό υπάρχει ήδη στο τρέχον ημερολόγιο. Τα διπλότυπα *δεν* αποκλείονται.
- **Επεξεργασία QSO** με το κουμπί *Επεξ.* στη γραμμή. Η φόρμα μεταβαίνει στη λειτουργία *Ενημέρωση QSO*, η γραμμή επισημαίνεται, και εμφανίζεται ένα κουμπί *Ακύρωση*. Η εναλλαγή ημερολογίου ή η διαγραφή του ακυρώνει αυτόματα την επεξεργασία.
- **Διαγραφή QSO** με το κουμπί *Διαγρ.* στη γραμμή (ζητά επιβεβαίωση).

## Εισαγωγή και εξαγωγή ADIF

- **Εξαγωγή**: κάντε κλικ στο *Εξαγωγή .adi* στην κεφαλίδα του ημερολογίου. Λαμβάνεται ένα αρχείο με `ADIF_VER 3.1.4` και `PROGRAMID local-qso` στην κεφαλίδα. Κάθε εγγραφή αντιστοιχίζει `CALL`, `QSO_DATE`, `TIME_ON`, `BAND`, `MODE`, `RST_SENT`, `RST_RCVD`.
- **Εισαγωγή**: κάντε κλικ στο *Εισαγωγή αρχείου .adi* κάτω από τη φόρμα δημιουργίας ημερολογίου και επιλέξτε ένα αρχείο `.adi`/`.adif`. Δημιουργείται ένα νέο ημερολόγιο με όνομα `Εισήχθη YYYY-MM-DD HH:MM UTC`. Η εισαγωγή δεν συγχωνεύεται ποτέ σε υπάρχον ημερολόγιο.
- Η μέτρηση μήκους πεδίου αντιμετωπίζεται ως αριθμός χαρακτήρων, που λειτουργεί για ASCII ADIF (όλα τα τυπικά πεδία QSO). Πολυμπαϊτό περιεχόμενο σε μη ουσιώδη πεδία κειμένου μπορεί να αναλυθεί παράξενα.

## Ιδιωτικότητα και δεδομένα

- Όλα τα δεδομένα αποθηκεύονται στο `localStorage` του περιηγητή κάτω από το κλειδί `local-qso:v1`.
- Τίποτα δεν μεταδίδεται πουθενά. Χωρίς backend, χωρίς κλήσεις API, χωρίς τηλεμετρία, χωρίς αναλυτικά.
- Η εκκαθάριση των δεδομένων του ιστότοπου, η χρήση ιδιωτικής/ανώνυμης λειτουργίας ή η χρήση διαφορετικού περιηγητή/συσκευής σημαίνει κενό ημερολόγιο — χρησιμοποιήστε το *Εξαγωγή .adi* για αντίγραφα ασφαλείας.

## Τεχνικές σημειώσεις

- Εφαρμογή μίας σελίδας, καθαρό HTML + CSS + JavaScript. Καμία πλαίσιο, κανένα build, καμία εξάρτηση.
- Αρχεία πηγής:
  - `index.html` — markup και meta tags.
  - `style.css` — θέματα και διάταξη (μεταβλητές ημέρας/νύχτας, mobile media queries).
  - `app.js` — κατάσταση, persistance, rendering, ADIF parser/serializer, αναζήτηση προθέματος διακριτικού → χώρα.
  - `favicon.svg` — ενσωματωμένο SVG favicon.
  - `manifest.webmanifest` — Web App Manifest (όνομα, χρώμα θέματος, scope, εικονίδιο) ώστε η εφαρμογή να είναι εγκαταστάσιμη ως PWA σε κινητό και επιφάνεια εργασίας.
  - `service-worker.js` — cache-first service worker που στην εγκατάσταση προ-φορτώνει κάθε στατικό αρχείο, στην ενεργοποίηση εξαλείφει τις παλιές κρυφές μνήμες και διατηρεί την εφαρμογή πλήρως εκτός σύνδεσης μετά την πρώτη επίσκεψη. Η εγγραφή παρακάμπτεται αυτόματα για το πρωτόκολλο `file://`, οπότε το άνοιγμα του `index.html` απευθείας από το δίσκο παραμένει καθαρό.
  - `i18n/<lang>.js` — ένα αρχείο μετάφρασης ανά υποστηριζόμενη γλώσσα (28 συνολικά). Καθένα είναι ένα μικρό IIFE που εκχωρεί στο `window.I18N[<lang>]` έναν επίπεδο χάρτη κλειδί→string. Το `t()` και το `applyLanguage()` στο `app.js` χειρίζονται τις αναζητήσεις (με αγγλικό fallback) και διασχίζουν το DOM ενημερώνοντας κάθε στοιχείο `[data-i18n*]`.
- Δοκιμασμένο σε πρόσφατες εκδόσεις Chromium, Firefox και Safari (επιφάνεια εργασίας + iOS).

## Αναγνωρίσεις

Κατασκευάστηκε από τον [YL3IM](https://www.qrz.com/db/YL3IM).
